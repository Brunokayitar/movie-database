# Movie Database

A movie search app using the OMDB API. Built with React and Tailwind CSS.

## Features
- Search for movies
- View movie details
- Responsive design

## Setup
1. Clone the repo
2. Run `npm install`
3. Get an API key from [OMDB](http://www.omdbapi.com/)
4. Create a `.env` file with `VITE_OMDB_API_KEY=your_key`
5. Run `npm run dev`

## Deployment
Deployed on Netlify/Vercel.
