# Movie Database

A React app that lets users search for movies and view details using the OMDb API.

## Features
- Search for movies by title
- View movie details (plot, cast, ratings, genre)
- Responsive design with Tailwind CSS

## Setup
1. Clone the repo
2. Run `npm install`
3. Get an API key from [OMDb](http://www.omdbapi.com/)
4. Create a `.env` file with `VITE_OMDB_API_KEY=your_key`
5. Run `npm run dev`

## Deployment
Deployed on Netlify/Vercel.

## Built With
- React
- Tailwind CSS
- React Router
- OMDb API
